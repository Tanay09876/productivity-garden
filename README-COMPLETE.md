## ğŸ› Troubleshooting

### Images Don't Show
**During Growth**: Normal! Emoji placeholders are used
**After Harvest**: Check browser console (F12) for errors

### Start Button Doesn't Work
- Clear browser cache (Ctrl+F5)
- Check that you're using the modified files
- Make sure JavaScript is enabled

### Progress Not Saving
- Check browser allows localStorage
- Don't use incognito/private mode
- Allow cookies for the site

### Plants Grow Too Fast/Slow
- Adjust `MIN_COMPLETE_TIME` in app.js line 15
- 5 minutes = `5 * 60 * 1000`
- 20 minutes = `20 * 60 * 1000`
